JavaScript Arrays

An array is a special variable, which can hold more than one value:-

It is a common practice to declare arrays with the const keyword:-

Input:-

           <p id="demo"></p>

    <script>
        const cars = ["VOLVO","BMW","SWIFT"];

        document.getElementById("demo").innerHTML = cars;
    </script>

Output:-

Another way to written arrays:-

Input:-

    <p id="same"></p>

    <script>
     const car = [];
     car[0] = "Fortuner";
     car[1] = "harrier";
 
     document.getElementById("same").innerHTML = car;
    </script>

Output:-

The toString() method returns an array as a comma separated string:-

Input:-

    <p id="fruits"></p>

    <script>
      const fruits = ["Banana", "Orange", "Apple", "Mango"];
  
      document.getElementById("fruits").innerHTML = fruits.toString();
    </script>

The length Property:-

The length property of an array returns the length of an array (the number of array elements).

Input:-

    <p id="fruit"></p>

    <script>
      const fruit = ["apple","coconat","orange"];
  
      document.getElementById("fruit").innerHTML = fruit.length
    </script>

Output:-
