CSS Text:-

CSS has a lot of properties for formatting text.

Text Color and Background Color

Input

        <p style="color: aqua; background-color: black;">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Deserunt, quasi.</p>
        <p style="color: black; background-color: bisque;">Lorem, ipsum dolor sit amet consectetur adipisicing elit. Illo, reprehenderit.</p>
        <p style="color: white; background-color: brown;">Lorem ipsum dolor sit amet consectetur adipisicing elit. Explicabo, suscipit.</p>
        <p style="color: white; background-color: blue;">Lorem ipsum dolor sit amet consectetur adipisicing elit. Eveniet, aliquam!</p>

Output

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Deserunt, quasi.

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Illo, reprehenderit.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Explicabo, suscipit.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Eveniet, aliquam!

CSS Text Alignment

Input

        <p style=" text-align: center;">Lorem ipsum dolor sit amet consectetur, adipisicing elit. Veritatis, nostrum.</p>
        <p style=" text-align: right">Lorem ipsum dolor sit amet.</p>

Output

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Veritatis, nostrum.

Lorem ipsum dolor sit amet.

Text Direction:-

The direction and unicode-bidi properties can be used to change the text direction of an element:

Input

    <p style="direction: rtl; unicode-bidi: bidi-override;">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Architecto, ratione.</p>

Output

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Architecto, ratione.

CSS Text Transformation:-

Input

    <p style="text-transform: lowercase;">Lorem ipsum dolor sit, amet consectetur adipisicing elit. Est, alias.</p>
    <p style="text-transform: capitalize;">Lorem ipsum dolor, sit amet consectetur adipisicing elit. Sunt, est!</p>
    <p style="text-transform: uppercase;">Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatum, quod.</p>

Output

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Est, alias.

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Sunt, est!

Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatum, quod.

CSS Text Shadow:-

The text-shadow property adds shadow to text.

Input

    <h1 style="text-shadow: 2px 2px;">Lorem ipsum dolor</h1>
    <h3 style="text-shadow: 2px 2px red;">Lorem ipsum dolor sit amet.</h3>

Output

Lorem ipsum dolor

Lorem ipsum dolor sit amet.
