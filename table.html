HTML tables allow web developers to arrange data into rows and columns.

Table

Input

        <style>
    table, th, td {
      border:1px solid black;
    }
    </style>
    
    <table style="width: 100%;">
    <caption>Student Details</caption>
    <tr>
        <th>student Name</th>
        <th>Roll No</th>
        <th>Moblie Number</th>
    </tr>
    <tr>
        <td>thid</td>
        <td>44</td>
        <td>4582252585581</td>
    </tr>
    <tr>
        <td>teferf</td>
        <td>55</td>
        <td>45541561324564</td>
    </tr>
    <tr>
        <td>dwdfd</td>
        <td>58</td>
        <td>54654512563545</td>
    </tr>
</table>

Output

Student Details
student Name Roll No Moblie Number
thid 44 4582252585581
teferf 55 45541561324564
dwdfd 58 54654512563545

Table Borders:-

The following values are allowed:-

  1. dotted
  2. dashed
  3. solid
  4. double
  5. groove
  6. ridge
  7. inset
  8. outset
