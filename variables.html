JavaScript Variables

JavaScript Variables can be declared in 4 ways:-

Automatically, Using var, Using let, Using const:-

1.Automatically:-

Input:-

        <p id="demo"></p>

        <script>
            x = 122
            y = 252
            z = x+y
            document.getElementById('demo').innerHTML = "This is value of z:" + z;
        </script>

Output:-

2.Using var:-

Input:-

        <p id="demo2"></p>

        <script>
            var x = 10;
            var y = 5;
            var z = x+y
            document.getElementById("demo2").innerHTML="This is value of z:" + z;
        </script>

Output:-

3.Using let:-

Input:-

    <p id="dome3"></p>

    <script>
     let x = 100;
     let y = 75;
     let z = x+y;
     document.getElementById("demo3").innerHTML="This is value of z:" + z;
    </script>

Output:-

4.Using const:-

Input:-

    <p id="semo"></p>

   
    <script>
     const p = 100;
     const o = 95;
     const s = x+y;
     document.getElementById("semo").innerHTML="This is value of s:" + s;
    </script>

Output:-

When to Use var, let, or const?

  1. Always declare variables
  2. Always use const if the value should not be changed
  3. Always use const if the type should not be changed (Arrays and Objects)
  4. Only use let if you can't use const
  5. Only use var if you MUST support old browsers.
