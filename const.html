JavaScript Const

The const keyword was introduced in ES6 (2015):-

  • Variables defined with const cannot be Redeclared and Variables defined with const cannot be Reassigned.

    • Input:-

                <p id="demo4"></p>

            <script>
                const name = ["Panner", "Chinese", "soup"];
                name[1] = ["Roti"];
                name.push = ["Ice-cream"];
                document.getElementById("demo4").innerHTML="Menu:-" + " " + name;
            </script>
