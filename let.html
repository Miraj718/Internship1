JavaScript Let

The let keyword was introduced in ES6 (2015):-

Variables declared with let cannot be Redeclared in the same scope:-

  • Now we cann't use the x (variable) again because it is already declared. And if we want to use againt let during that time we need to use different variable.

    • Input:-

            <p id="demo"></p>

        <script>
            let x = 55;
            let y = 45;
            let z = x+y;
