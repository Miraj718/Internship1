JavaScript Operators

Javascript operators are used to perform different types of mathematical and logical computations:-

  1. The Assignment Operator = assigns values
  2. The Addition Operator + adds values
  3. The Multiplication Operator * multiplies values
  4. The Comparison Operator > compares values

1.The Assignment Operator = assigns values:-

Input:-


    <p id="dome"></p>

    <script>
        let a = 15;
        let b = 25;
        let c = a+b;
        document.getElementById("dome").innerHTML=c;
    </script>

Output:-

Input:-


     <p id="demo"></p>

     let x = 5;
     <script>
     let z = x + y;
     let y = 2;
     </script>
     document.getElementById("demo").innerHTML = z;

3.The Multiplication Operator * multiplies values:-

Input:-


    <p id="demos"></p>

    <script>
        let q = 5;
        let w = 2;
        let e = q * w;
        document.getElementById("demos").innerHTML = e;

    <p id="demop"></p>

        document.getElementById("demop").innerHTML = "Is A less than B? " + result;
