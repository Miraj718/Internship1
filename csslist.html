HTML lists allow web developers to group a set of related items in lists.

Input


        <head>
        <style>
        ol {
          background: #ff9999;
          padding: 20px;
        }
        
        ul {
          background: #3399ff;
          padding: 20px;
        }
        
        ol li {
          background: #ffe5e5;
          color: darkred;
          padding: 5px;
          margin-left: 35px;
        }
        
        ul li {
          background: #cce5ff;
          color: darkblue;
          margin: 5px;
        }
        </style>
        </head>


        
    <h2>An Unordered HTML List</h2>

    <ul>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
    </ul>

    <h2>An Ordered HTML List</h2>

    <ol>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
    </ol>

    <h2>Other list</h2>

Output

An Unordered HTML List

  • Coffee
  • Tea
  • Milk

An Ordered HTML List

  1. Coffee
  2. Tea
  3. Milk
