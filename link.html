html link:-

HTML links are defined with the a tag:

Input

<a href="https://www.youtube.com"> Go to youtube </a>

Output

Go to youtube

html image:-

Html images are defined with the img tag :-

Input

<img src="./image/Nature.jpg" alt="Nature image" width="104" height="142">

Output

The target Attribute:-

  • _self - Default. Opens the document in the same window/tab as it was clicked
  • _blank - Opens the document in a new window or tab
  • _parent - Opens the document in the parent frame
  • _top - Opens the document in the full body of the window

Input

<a href="https://www.youtube.com" target="_blank">Go to youtube</a>

Output

Go to youtube

Absolute URLS vs. Relative URLS:-

Absolute URLs

Input

<p><a href="https://www.w3.org/">W3C</a></p>
<p><a href="https://www.google.com/">Google</a></p>

Output

W3C

Google

Relative URLS

Input

<p><a href="/image/FAQS.jpg">FAQS</a></p>
<p><a href="./image/Nature.jpg">Nature Images</a></p>

Output

Nature Images

FAQS

Link to an Email Address:-

Input

<a href="mailto:someone@example.com">Send email</a>

Output

Send email

Button as a Link:-

To use an HTML button as a link, need to add some JavaScript code.

Input

<button onclick="window.location.href='https://www.youtube.com/'">GO to youtube</button>

Output

Link Titles:-

The title attribute specifies extra information about an element.

Input

<a href="https://www.youtube.com/" title="Go to youtube for watch some videos "> visit youtube for entertainment </a>

Output

visit youtube for entertainment
