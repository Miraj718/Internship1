CSS Links:-

With CSS, links can be styled in many different ways.

Styling Links

Input

    <h2>Style a link with a color</h2>
    <p><b><a href="https://www.youtube.com/" target="_blank">This is a link</a></b></p>

Output

Style a link with a color

This is a link

Background Color:-

Input

        <head>
<style>
        a:link{
            background-color: yellow;
        }
        a:hover{
            background-color: aqua;
        }
        a:visited{
            background-color: brown;
        }
    </style>
</head>
        <p><b><a href="https://www.youtube.com/" target="_blank">This is a link</a></b></p>

Output

This is a link
