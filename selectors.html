The CSS element Selector

The element selector selects HTML elements based on the element name.

Input


     <head>
       <style>
      p {
           color: red;  
          } 
       </style>
      </head>
      <p style="color: red;">Every paragraph will be affected by the style.</p>
      <p style="color: red;">Me too!</p>
      <p style="color: red;">And me!</p>

Output

Every paragraph will be affected by the style.

Me too!

And me!

The CSS id Selector:-

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

Input


    <head>
    <style>
    #xyz {
      color: blue;
    }
    </style>
    </head>
    <p id="xyz">css help to save lot of time for developers </p>

Output

css help to save lot of time for developers

The CSS class Selector:-

To select elements with a specific class, write a period (.) character, followed by the class name.

Input


        <head>
<style>
.xyz2 {
  color: grey;
}
</style>
</head> 

<h1 class="xyz2">there are many fun videos on youtube for reduce stress</h1>

Output

there are many fun videos on youtube for reduce stress

The CSS Universal Selector:-

The universal selector (*) selects all HTML elements on the page.

Input

    <head>
<style>
* {
  text-align: center;
  color: blue;
}
</style>
</head>
<h1>this is me </h1>
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Nulla, quia?</p>

Output

this is me

Lorem ipsum dolor sit amet consectetur adipisicing elit.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Nulla, quia?
