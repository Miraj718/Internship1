Bootstrap 5 Text

Bootstrap 5 uses a default font-size of 1rem (16px by default), and its line-height is 1.5.

1.<h1> - <h6>

Bootstrap 5 styles HTML headings (<h1> to <h6>) with a bolder font-weight and a responsive font-size.

Input

        <p>The font-size of each Bootstrap heading depends on the screen size. Try to resize the browser window to see the effect.</p>
        <h1>h1 Bootstrap heading</h1>
        <h2>h2 Bootstrap heading</h2>
        <h3>h3 Bootstrap heading</h3>
        <h4>h4 Bootstrap heading</h4>
        <h5>h5 Bootstrap heading</h5>
        <h6>h6 Bootstrap heading</h6>

Output

The font-size of each Bootstrap heading depends on the screen size. Try to resize the browser window to see the effect.

h1 Bootstrap heading

h2 Bootstrap heading

h3 Bootstrap heading

h4 Bootstrap heading

h5 Bootstrap heading
h6 Bootstrap heading

2.<small>

In Bootstrap 5 the HTML element (and the .small class) is used to create a smaller, secondary text in any heading:

Input

        <h1>this is nowadays many people</h1>
        <h2>this is <small>nowadays many people</small></h2>
        <h3>this is <small>nowadays many people</small></h3>
         <h4>this is <small>nowadays many people</small></h4>

Output

this is nowadays many people

this is nowadays many people

this is nowadays many people

this is nowadays many people

3.<mark>

Bootstrap 5 will style <mark> and .mark with a yellow background color and some padding:

Input

 <p>use the mark element to <mark>highlight</mark> text</p>

Output

use the mark element to highlight text

3.Keyboard Inputs

Input

<p>Use <kbd>ctrl + p</kbd> to open the Print box.</p>

Output

Use ctrl + p to open the Print box.
